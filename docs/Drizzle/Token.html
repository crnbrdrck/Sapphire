    • Drizzle::Token

    Overview

    A Token is generated by the Lexer as it reads in Drizzle source code.

    Tokens are then used to generate an Abstract Syntax Tree (AST) which can then be used to actually make the language do things.

    Defined in:

    drizzle/token.cr
